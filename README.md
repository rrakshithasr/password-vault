# 🔐 Password Vault (Python)

A simple and secure command-line Password Vault built with Python. This application allows users to store, search, update, and manage website credentials in a JSON file. It also includes a strong random password generator.

---

## 📌 Features

* ➕ Add new website credentials
* 👀 View all saved accounts
* 🔍 Search for a saved account
* ✏️ Update an existing password
* 🗑️ Delete saved credentials
* 🔑 Generate strong random passwords
* 💾 Automatically save data in a JSON file
* 🧱 Object-Oriented Programming (OOP) design
* 📂 Single-file implementation

---

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Save the program as `password_vault.py`.
3. Open a terminal or command prompt in the project folder.
4. Run the program:

```bash
python password_vault.py
```
